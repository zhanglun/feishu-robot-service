# Feishu robot service

a simple node service based on Nest.js for aiming push feishu message to any group

## How to use 

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# debug mode
$ pnpm run start:debug

# production mode
$ pnpm run start:prod
```

## Building

```bash
$ pnpm run build
```
